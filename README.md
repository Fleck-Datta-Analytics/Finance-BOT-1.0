# BOT Finance - Fleck Finance Group

Este projeto é um algoritmo de negociação automatizado desenvolvido pelo Fleck Finance Group. O BOT utiliza médias móveis para tomar decisões de compra, venda ou manutenção para ações específicas. O projeto foi desenvolvido em Python e utiliza bibliotecas como `yfinance`, `pandas`, `numpy` e `matplotlib` para análise de dados e visualização.

## Funcionalidades

O BOT Finance possui as seguintes funcionalidades:

- Utiliza médias móveis para análise técnica de ações.
- Calcula probabilidades de lucro e tamanhos ótimos de posição usando o Critério de Kelly.
- Executa ordens de compra e venda de ações.
- Fornece visualizações gráficas dos preços históricos, médias móveis e pontos de compra/venda.

## Como Usar

1. Clone este repositório para o seu ambiente local.
2. Certifique-se de ter as bibliotecas necessárias instaladas (yfinance, pandas, numpy, matplotlib).
3. Execute o código do BOT em um ambiente Python (por exemplo, Jupyter Notebook).
4. O BOT solicitará um símbolo de ticker da ação para análise. Insira o símbolo da ação desejada.
5. O BOT calculará médias móveis, tomará decisões de negociação e fornecerá resultados.
6. Para sair, digite "exit" quando solicitado a inserir o símbolo de ticker.
